# Workflow

**Pașii standard de lucru cu Codex**
1. Citește `docs/PROJECT_CONTEXT.md` și `docs/TASKS.md`.
2. Alege un task clar și verificabil.
3. Stabilește fișierele afectate și criteriile de finalizare.
4. Implementează schimbările incremental și verifică rezultatul.
5. Actualizează `docs/TASKS.md` și, dacă e cazul, `docs/DECISIONS.md`.

**Cum pornești un task nou**
1. Adaugă task-ul în `docs/TASKS.md` cu status `TODO`.
2. Completează scop, fișiere afectate și DoD.
3. Creează un prompt folosind `prompts/NEW_TASK.md`.

**Cum finalizezi un task**
1. Verifică DoD-ul.
2. Marchează task-ul `DONE` în `docs/TASKS.md`.
3. Dacă ai luat o decizie nouă, adaug-o în `docs/DECISIONS.md`.

**Regulă de lucru**
- Un task = un rezultat verificabil.
